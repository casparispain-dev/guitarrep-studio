# GuitarRep Studios — Live Site

## Project
GuitarRep Studios (GRS) holding site deployed via Cloudflare Pages at guitarrep.com.

## Files
- index.html — GuitarRep master holding page
- chordrep.html — ChordRep holding page  
- alpha.html — ChordRep Alpha v1 tester build
- about.html — About page
- legal.html — Legal page
- robots.txt — SEO
- assets/ — Images and fonts

## Design system
- Background: #F7EAED
- Rose: #C45A6E / Burgundy: #993556
- Display font: Brown Carolina
- Layout: two-column Scandinavian minimal

## Deploy
Push to GitHub → Cloudflare Pages auto-deploys from main branch.

## Stack
HTML/CSS only. No build tools. No frameworks.
